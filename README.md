# Meaningful Bot

A discord bot to hold polls and save essential messages.

### Commands

#### !poll

This command is used to create a poll. The bot will ask 3 questions - 

1. Question [String] [Coffee or Tea]
2. Options [Array] [Coffee, Tea, None]
3. Time [Integer] [5]
